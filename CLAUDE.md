# 점심 메뉴 추천해줘!

## 프로젝트 설명
오늘의 기분과 최근에 먹은 음식을 기반으로 오늘 먹을 점심 메뉴를 추천해주는 웹 애플리케이션입니다.

## 사용 기술
- **HTML**: 웹 페이지 구조
- **CSS**: 디자인 및 스타일링
- **JavaScript**: 메뉴 추천 로직 및 상호작용

## 파일 구조
```
프로젝트 폴더/
├── index.html          # 메인 HTML 파일
├── style.css           # 스타일시트
├── script.js           # JavaScript 로직
└── CLAUDE.md           # 프로젝트 문서
```

## 실행 방법
1. 프로젝트 폴더에서 `index.html` 파일을 찾습니다.
2. 웹 브라우저(Chrome, Firefox, Safari 등)에서 `index.html` 파일을 엽니다.
3. 화면의 안내에 따라 기분과 최근 식사 정보를 입력합니다.
4. 추천 버튼을 클릭하여 점심 메뉴를 확인합니다.

## 코딩 규칙
1. **한국어 주석 사용**: 모든 코드에 한국어로 된 설명 주석을 작성합니다.
2. **초보자 친화적**: 복잡한 구조보다는 읽기 쉽고 이해하기 쉬운 코드를 작성합니다.
3. **상세한 설명**: 각 함수, 변수, 로직마다 설명 주석을 달아 코드의 목적을 명확히 합니다.
4. **명확한 변수명**: 변수와 함수의 이름은 그 목적을 명확히 알 수 있도록 작성합니다.

## 개발 가이드
- 코드를 수정할 때는 주석을 먼저 읽어 해당 부분의 역할을 이해합니다.
- 새로운 기능을 추가할 때는 반드시 주석을 함께 작성합니다.
- 테스트는 브라우저에서 직접 실행하여 확인합니다.
